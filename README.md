# LoL Champion Skill Quiz

리그 오브 레전드 챔피언의 스킬 설명을 보고 챔피언 이름을 맞히는 C 기반 콘솔 퀴즈 게임입니다. (Windows)

데모 영상(YouTube): https://youtu.be/jMwWmPeBHlk

## 개요

챔피언의 스킬 설명을 출력하고 사용자가 이름을 맞히는 콘솔 퀴즈 게임입니다. 난이도 모드를 지원하며 정답·오답 기록을 저장해 종료 시 보여줍니다.

## 주요 기능

- 3단계 난이도: Easy(패시브+Q/W/E/R), Normal(Q/W/E), Hard(패시브)
- 문제 수 선택(10/30/50), 매 실행마다 챔피언 순서 셔플
- 정답·오답 목록 출력 (연결 리스트로 저장)

## 동작 방식

```
[난이도 선택] → [문제 수 선택] → [스킬 설명 출력] → [이름 입력]
→ ... 반복 ... → [정답/오답 목록 출력]
```

## 기술 스택

C · Windows · 연결 리스트(Linked List)
